API Basejump: URL Shortener

  • Pass a URL as a parameter and receive a shortened URL in the JSON response.
  • When you visit that shortened URL, it will redirect you to the original URL you passed in as a parameter.

Example creation usage:

https://url-shortener-ms-asell22.herokuapp.com/new/https://www.google.com

Example creation output:

{ "original_url": "https://www.google.com", "short_url": "https://url-shortener-ms-asell22.herokuapp.com/1" }

Click on the "short_url" to be directed to the "original_url"
